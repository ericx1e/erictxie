Web Projects

word bord

A daily word puzzle with Rubik's cube inspired movement. I made an api to manage generating the puzzle and storing the daily leaderboards. Using Java, I wrote a greedy algorithm to find high scores for each day and used the solver to generate fun boards.

24

24 is a math card game I played with my family growing up. The rules are simple: use all four cards and basic math operations to make 24. When using playing cards, there isn't always a solution. My version includes 1700 lines of code to check all 7680 permutations of numbers, operations, and parathases to check and solve any set of numbers. There is also confetti :)

3body simulation

A recreation of the 3body game from The Three-Body Problem by Cixin Liu. I used Unity to build the world and C# scripts create the first-person simulation. I recorded a 3D video in Unity so the world can be experienced using VR glasses.

prime bord

A tile sliding game inspired by 2048. Instead of only powers of 2, tiles consist of all whole numbers. Tiles with the same value can combine, but prime numbers can combine with any number! As the game progresses, prime numbers naturally become more sparse and the game becomes harder.

galton board

A galton board simulation made using the matter.js physics engine. Creates a graph based on balls fall out of the board which gradually forms the normal curve, proving the central limit theorem.
